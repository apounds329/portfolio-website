00 context

This was the final project for a 2 month specialization course in Frontend Development for Designers by CareerFoundry. This responsive portfolio website was designed to showcase all my UX|UI design projects by demonstrating my newly acquired frontend development skills.

The Challenge

My main challenge was having no prior experience with frontend development. However, I was movtivated with curiosity about coding and web design and appreceiated the creative freedoms it provided me as a designer.

Designers and devlopers are like vampires & werewolves- for some reason, they don't hang out much and inevitably start to speak different languages over time despite both being supernatural creatures of the night! Initially torn between the UI specialization (my comfort zone) and Frontend development (new and scary)selections, I decided to cannon-ball into the world of development propelled by strong desire as a designer to communicate ideas effectively with developers while also being able to create design decisions exactly how I imagined them.

The Goals

As a UX|UI designer, I really desired to showcase my research and design work in a way that would catch attention and keep it. Designing a portfolio with a mobile first approach will allow access across technological platforms. I needed a way to represnt someone creative & fun, yet professional and knowledgable. I wanted to leave room to breathe, but also include elements of interactivity to engage and delight visitors.

01 Process

User flow

I started designing with mobile-first methodology in mind, making sure I coded appropriately for breakpoints between mobile, tablet, and desktop. I examined other UX|UI designer and creative portfolios across the web. 14Islands, Free Faces font gallery, and Dog stuido were all sources of inspiration for navigation and layout.

02 Design

Colour & Typography

I chose colours that were clean yet subtly warm, with accents of soft neons to spark curiosity. The colours are accessible and friendly. I drew inspiration from early fifteenth-centry Italian scripts of Renaissance Humanism.

Fonts

Maiola is a serf font, very stylized and classic which is perfect for inspiring that old-school charm. Since my logo is simple, use of this font inspires a theme on its own.


Heebo is a sans-serif font that ensures readability and modern simplicity. It also looks great as a weighted heirarchy.

|wireframes

Low-fidelity

I quikly iterated some low-fi wireframes to outline my thoughts for navigation and structure.

03 CODING

HTML/CSS/JavaScript

After deciding on the style of my website, I began to code the different pages of my site using the text editor Atom. Keeping the DOM (document object model) in mind began with the basic HTML structure of an index.html and about.html. I then added a style.css and scripts.js folder to create style and interactivity. The code was validated and checked with linters to ensure quality.

Accessibility

I checked the font and background colour ratios using W3C Validator. I also implemented alt tags for all images in case a screen reader is used.

Mobile first

The inspect tool on Chrome was my best friend. I could easily check to make sure my designs transitioned from mobile to tablet to laptop.
